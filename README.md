## ✨ Aesthetic Todo App

A simple and aesthetic Todo List web application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Add tasks
- Delete tasks
- Mark tasks as completed
- Responsive glassmorphism UI
- Smooth animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

Aesthetic-Todo-App/
│
├── index.html
├── style.css
├── script.js
└── README.md

## 📸 Preview

Beautiful pastel gradient UI with glassmorphism effect.

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
